# Ⅷ. 学習（Epistemic Loop）：無常に抗う“プロセスの恒常化”

## 1) Upāya Loop（方便ループ）

- 観（Dの洞察）→ 意図（軽減仮説）→ 最小実験 → 省察（データ＋対話）→ 誓願更新
- 無常は“正解の更新”を強制する  
  → **プロセスを習慣化**して、変化に耐える

## 2) 八正道スコアカード（意思決定の品質保証）

- **正見**（データに裏付く現実把握）
- **正思惟**（無害性・十分性の遵守）
- **正語**（誠実な表現）
- **正業**（施策の行為倫理）
- **正命**（生業そのものの適正）
- **正精進**（改善速度）
- **正念**（振り返りと透明な記録）
- **正定**（静けさ：意思決定の雑音を減らす制度）

> **重要**：これは“研修メニュー”ではなく統治の一部。スコアが閾値未満なら可決を保留する
